# Prompts Directory

This directory contains all prompt templates used in the Secret Hitler LLM game.

## Files

- `base_prompts.json`: Core system and decision prompts
- `README.md`: This file

## Prompt Structure

Prompts are organized by:

1. **System prompts**: Define the role and objectives for each player type
2. **Decision templates**: Templates for different decision types
3. **Context formatters**: How to present game state to LLMs

## Customization

You can modify prompts to:

- Change LLM behavior and strategy
- Add more detailed instructions
- Experiment with different persuasion techniques
- Test various levels of deception

## Best Practices

- Keep prompts clear and unambiguous
- Provide sufficient game context
- Include examples when helpful
- Balance instruction detail with creativity
